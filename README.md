# Idena telegram bot

Install all dependencies

`npm i && npm i -g nodemon`

Start the local server

`npm run dev`

Run project

```
npm run start
```

## Env variables

```
FAUNADB_SECRET=
BOT_TOKEN=
WEBSERVER_PORT=
SIGNIN_CALLBACK=
SIGNIN_NONCE_ENDPOINT=
SIGNIN_AUTH_ENDPOINT=
NODE_URL=
NODE_KEY=
```
